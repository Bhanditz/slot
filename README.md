# Slot Machine
slot machine game, built using  vanilla HTML, CSS and JavaScript. 


## Installation, Build & Deployment
1) Clone repository
2) Run `npm install`
    - *Development*: run `npm start` and go to `http://localhost:8080`
    - *Production*: run `npm run build` and serve from `/dist`
